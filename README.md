# Movie Portal

This project is a **Movie Portal built with React** that uses the TMDB API to display movies.

---

## Features

- Search for movies by title
- Filter movies by genre
- Pagination for movie lists
- Display top-rated movies
- Display movie details in a Modal (React Portal)
- Single Page Application (SPA) with React Router
- Styled using CSS Modules

---

## Unit Testing

Unit tests were implemented using **React Testing Library**.

- Tested components: MovieCard, MovieDetails, Home, App
- Tested user interactions (click, search, pagination)
- Mocked API calls
